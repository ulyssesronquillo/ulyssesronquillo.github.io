Gmail Pop Server

Feb 8, 2006 • ulysses

I’ve been trying to configure my email client to work with Gmail using a POP server a couple days ago. I haven’t had any success until this morning. I even read the GMail Online Help file. It tells a user to configure the POP server to pop.gmail.com.

Well, I found out the POP server does not work. I saw a couple of people make an online suggestion to change the POP server to pop.googlemail.com. I tried it, and it works! I hope Google makes that correction to their help file because it’s confusing many people.

Unfortunately, Google Mail does not support IMAP. I’m not privy to the reasons why they haven’t implemented it yet since IMAP has many advanced features that’s better than POP.

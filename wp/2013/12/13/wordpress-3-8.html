Wordpress 3.8

Dec 13, 2013 • Ulysses

WordPress 3.8 was released today. It’s named “Parker” after the jazz saxophonist Charlie “Bird” Parker. WordPress 3.8 has a new modern, clean, aesthetic Admin Dashboard. The clean design makes it easier to navigate and read the Admin Dashboard. It comes in 8 stunning color themes. My favorite is “Midnight.”

In addition, WordPress now uses the “Open Sans” typeface which is friendly for desktop and mobile viewing. WordPress 3.8 now uses vector-based icons that scales to every device. To top it all off, WordPress 3.8 comes with a new sleek magazine-styled default theme called “Twenty Fourteen.”

If you are new to CMS, blogging or just want to start a website, consider using WordPress. It’s open and it’s free. Not only that, it also now powers 20% of the Internet worldwide.

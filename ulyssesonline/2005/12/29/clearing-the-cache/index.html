Clearing the Cache

Dec 29, 2005 • ulysses

One of the nice features of Firefox 1.5 is having the ability to clear off private data each time you close your browser. You have this option if you are using Firefox 1.5. You can delete browsing history, forms, passwords, download history, cookies, cache and authenticated sessions.

In my case, I delete all the listed items above each time I close my browser. It’s a nice feature especially since some websites are using cookies to track your web experience.

To clear the cache, pull down the Tools menu and select Options. Click the Privacy button and click the Settings button located on the bottom. Select “Clear Private Data when closing Firefox.” Finally, select which items you want cleared. That’s it. Happy safe browsing.

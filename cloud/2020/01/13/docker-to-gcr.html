docker to gcr

Jan 13, 2020 • ulysses

Here’s a couple of steps to move a Docker image to the Google Container Registry.

Create a container image.

docker build -t gcr.io/${PROJECT_ID}/my-app:v1 .

Push image to GCR.

docker push gcr.io/[PROJECT-ID]/[IMAGE]
